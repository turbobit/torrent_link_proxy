Transmission URL Supporter

마그넷 링크와 토렌트 파일을 Transmission 서버에 쉽게 업로드하세요

버전 1.0.0

다운로드

Chrome 확장 프로그램을 설치하려면 아래 링크를 클릭하세요:

ZIP 파일 다운로드

크롬 확장 프로그램 개발자 모드에서 설치 가능

주요 기능

  • 마그넷 링크 우클릭으로 Transmission 서버에 직접 업로드
  • 토렌트 파일 드래그 앤 드롭 지원
  • Transmission 서버 설정 저장
  • 업로드 상태 알림
  • 간편한 옵션 페이지로 서버 설정

설치 방법

  1. 위의 "ZIP 파일 다운로드" 버튼을 클릭하여 확장 프로그램을 다운로드하세요.
  2. Chrome 브라우저에서 chrome://extensions/로 이동하세요.
  3. 우측 상단의 "개발자 모드"를 활성화하세요.
  4. 다운로드한 ZIP 파일을 압축 해제하세요.
  5. "압축해제된 확장 프로그램을 로드합니다" 버튼을 클릭하고 압축 해제한 폴더를 선택하세요.
  6. 확장 프로그램이 설치됩니다.

사용 방법

  1. 확장 프로그램 아이콘을 클릭하여 옵션 페이지를 열어 Transmission 서버 정보를 설정하세요.
  2. 마그넷 링크를 우클릭하고 "Transmission에 업로드"를 선택하세요.
  3. 토렌트 파일을 브라우저에 드래그하여 업로드하세요.

기술 정보

  • Manifest 버전: 3
  • 권한: contextMenus, storage, activeTab, notifications
  • 호스트 권한: 모든 URL
  • 배경: Service Worker (background.js)
